# Deskripsi Aplikasi
PakeLagi App adalah aplikasi berbasis mobile yang berfungsi sebagai platform jual-beli produk fashion preloved (bekas pakai). Aplikasi ini dirancang untuk memudahkan masyarakat dalam membeli, menjual, dan mencari produk fashion yang sudah tidak digunakan dengan harga murah. Tagline aplikasi ini adalah "Pintar Pilih, Pintar Gaya!"

# Tools yang Digunakan
- Figma: Untuk membuat desain awal dari aplikasi
- VSCode: Untuk membuat struktur kode
- PHP: Untuk memproses data dari database
- MySQL: Untuk membuat database
- XAMPP: Sebagai penghubung server database

# Alur Penggunaan Aplikasi
1. User membuka aplikasi PakeLagi
2. User membuat akun (register/login)
3. User melihat produk yang ada di beranda aplikasi
4. User mencari produk yang diinginkan/dibeli
5. User memfilter pencarian produk berdasarkan kategori
6. User memasukkan produk yang dipilih ke dalam keranjang
7. User men-check out produk yang dipilih
8. User melakukan pembayaran

# Fitur-Fitur Aplikasi
a.	Fitur untuk semua pengguna (User):
    •	Registrasi dan login akun (via email/nomor HP) 
    •	Logout 
    •	Lihat dan edit profil pribadi

b.	Fitur untuk Pembeli (Buyer):
    •	Lihat katalog produk preloved 
    •	Cari produk berdasarkan kata kunci 
    •	Filter produk berdasarkan kategori (baju, celana, sepatu, aksesoris, dll) 
    •	Filter produk berdasarkan range harga 
    •	Lihat detail produk lengkap (gambar, deskripsi, harga, info seller) 
    •	Hubungi seller melalui WhatsApp

c.	Fitur untuk Penjual (Seller):
    •	Upload produk (foto, deskripsi, harga, kategori) 
    •	Edit dan hapus produk 
    •	Kelola daftar produk yang diunggah
