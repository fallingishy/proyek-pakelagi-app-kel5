# Deskripsi Aplikasi
## PakeLagi App adalah aplikasi berbasis mobile yang berfungsi sebagai platform jual-beli produk fashion preloved (bekas pakai). Aplikasi ini dirancang untuk memudahkan masyarakat dalam membeli, menjual, dan mencari produk fashion yang sudah tidak digunakan dengan harga murah. Tagline aplikasi ini adalah "Pintar Pilih, Pintar Gaya!"
